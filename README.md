# java-desktop-app-starter

A starter template for JavaFX projects.

### Dependencies
---

- [Java 11][1]
- [Maven][2]
- [JavaFX][3]

### Run with the Maven plugin
---

```
$ mvn clean javafx:run
```

[1]:https://adoptopenjdk.net/?variant=openjdk11&jvmVariant=hotspot
[2]:https://maven.apache.org
[3]:https://openjfx.io
