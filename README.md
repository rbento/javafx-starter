# javafx-starter

A template for desktop applications with JavaFX.

### Dependencies
---

- [Java 18][1]
- [Maven][2]
- [JavaFX][3]

### Run with the JavaFX Maven [plugin][4]
---

```
$ mvn clean javafx:run
```

[1]:https://www.oracle.com/java/technologies/downloads/#java18
[2]:https://maven.apache.org
[3]:https://openjfx.io
